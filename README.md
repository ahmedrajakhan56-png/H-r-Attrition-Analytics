 Project Overview

This project analyzes employee attrition using the IBM HR Analytics dataset. The goal is to identify key factors driving attrition and provide actionable recommendations to reduce employee turnover.

Dataset: IBM HR Analytics Dataset (1,470 employees, 35 features)  
Duration: 3 Days  
Tools Used: Python, SQL Server, Power BI, AWS Cloud  

 Business Problem

The company is experiencing a **16.1% annual attrition rate**, losing **237 employees** per year. With a replacement cost of **₹5 Lakhs per employee**, this results in an annual loss of **₹11.85 Crores**. The project aims to reduce attrition by 15%, saving **₹1.78 Crores annually**.

 Technology Stack

| Tool | Purpose |
|------|---------|
| Python | Data cleaning, feature engineering, EDA |
| SQL Server | Data storage, querying, analysis |
| Power BI | Interactive dashboards (3 pages) |
| AWS S3 | Cloud data storage |
| AWS Glue | Data cataloging |
| AWS Athena | SQL queries on cloud data |

 Key Findings

| Insight | Finding |
|---------|---------|
| Attrition Rate | 16.1% (237 employees) |
| Annual Loss | ₹11.85 Crores |
| Top Driver | Overtime (2.7x higher risk) |
| Salary Impact | <20K band: 31.8% attrition |
| New Hires | <1 year: 35.1% attrition |
| High Performers | 230 employees (3.9% attrition) |
